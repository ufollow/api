## Sobre a API

### How to debug locally

Before you start:

- Install [.NET Core SDK](https://www.microsoft.com/net/download/windows/)
- Install [PostgreSQL](https://www.postgresql.org/)
- Install [VS Code](https://code.visualstudio.com/)
